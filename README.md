# 📈 종가매수 추천 시스템

하바로셀 5대 매매 원칙 + 차트 27장 분석을 통합한 종가매수 종목 자동 추천 시스템.

## 핵심 기능

- **오늘의 종가매수 추천 TOP N** (실시간 KRX 데이터)
- **2020~현재 백테스트** (일별 추천 + 다음날 수익률 자동 계산)
- **완전 커스터마이징** (필터/가중치/임계값 슬라이더 UI)
- **CSV 결과 다운로드** (매매일지 작성용)

## 5대 시그널 (점수화)

| 시그널 | 설명 | 기본 가중치 |
|---|---|---|
| **S1** 박스권 돌파 | 직전 60일 횡보 박스 상단 돌파 = "힘의 응축" | 30 |
| **S2** 거래량 폭증 | 20일 평균 × 3배 이상 = "수급 선점" | 25 |
| **S3** 장대양봉 | 시가→종가 +5% 이상 (윗꼬리 페널티) | 15 |
| **S4** 이평선 정배열 | 종가 > MA5 > MA20 > MA60 | 15 |
| **S5** 전고점 돌파 | 직전 60일 고가 돌파/95%+ 근접 | 15 |

총점 = 가중 평균 (0~100). 기본 추천 최소 점수 40.

## 기본 필터 (사용자 변경 가능)

- 거래대금 ≥ **50억원**
- 시가총액 ≥ **2,000억원**
- 등락률 **7~29%** (상한가 30% 제외)
- KOSPI + KOSDAQ (체크박스로 선택)
- ETF/ETN/SPAC/리츠 자동 제외


## 사용법

### 탭 1: 오늘의 추천
1. 좌측 사이드바에서 필터/가중치 조정 (또는 기본값 그대로)
2. **"🔍 스캔 실행"** 클릭
3. 1~3분 후 TOP N 종목 + 시그널별 점수 표시
4. 3:00 PM 이후 실행하면 당일 종가 기준 추천

### 탭 2: 백테스트
1. 기간 설정 (시작일/종료일)
2. 매도 전략 선택:
   - `next_open`: 다음날 시초가 (보수적)
   - `next_high`: 다음날 장중 고가 (이상치, 최선의 경우)
   - `next_close`: 다음날 종가
   - `t2_close`: T+2일 종가 (2일 보유)
3. 유니버스 크기 조정 (300 = 시총 상위 ~300위, 빠름 / 2000 = 전체, 느림)
4. **"🚀 백테스트 실행"** 클릭
5. 결과:
   - 통계 (평균 수익률, 승률, 샤프 비율 등)
   - 일자별 추천 + 수익률 테이블
   - 월별 평균 수익률 차트
   - CSV 다운로드 버튼

### 탭 3: 시그널 설명
원칙 및 룰 요약 (참고용)

## 커스터마이징 예시

### 1. "보수적 추천만" (점수 높은 종목만)
- 최소 점수 → **60**
- 거래대금 ≥ **100억**
- 시가총액 ≥ **5,000억**

### 2. "박스권 돌파 위주"
- S1 가중치 → **50**
- S2 가중치 → **30**
- S3~S5 → 각 **5~10**

### 3. "급등 추격형" (위험 ↑)
- 등락률 최소 → **15%**
- S2 (거래량) 가중치 → **40**
- S3 (장대양봉) → **30**

## 데이터 출처

- **FinanceDataReader** (오픈소스, MIT 라이센스)
- KRX 공식 데이터 기반
- KOSPI ~948개 + KOSDAQ ~1,822개 = 약 2,770개 종목

## 한계 및 주의사항

⚠️ **이 도구는 통계적 보조일 뿐, 매매 보장 아님**

1. **선택 편향**: 분석에 사용된 27장 차트는 성공 사례 위주. 실제 적중률은 50~60% 추정.
2. **데이터 지연**: 장 마감 후 데이터 업데이트까지 최대 30분 지연 가능.
3. **거래대금 백테스트 근사**: `종가 × 거래량`으로 추정 (실제 거래대금과 약간 차이).
4. **상장폐지 종목**: FinanceDataReader가 현재 상장 종목 위주라 백테스트 시 생존 편향 가능.
5. **3:20 PM 수급 분석 미포함**: 하바로셀이 강조하는 동시호가 수급은 별도 HTS 모니터링 필요.
