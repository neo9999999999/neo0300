# 📖 종가매매 전략 가이드

> 하바로셀 + 하승훈 통합 — 8대 원칙 + 5단계 프로세스 + 3대 패턴

---

## 0. 종가매매란?

**정의**: 장 마감 직전(오후 2:30~3:30)에 매수하여 다음 날 시초가 또는 장중 고가에서 매도하는 1~2일 단타 전략.

**핵심 가치**:
- 장중 변동성(노이즈) 제거 — 일봉 완성형 확인 후 진입
- 시간 효율 — 30분~1시간 집중 분석으로 의사결정
- 가짜 돌파(Bull Trap) 방어 — 윗꼬리 음봉 회피
- 데이터 기반 — 종가는 모든 정보가 압축된 결산점

---

## 1. 8대 분석 원칙 [캔·거·이·추·지·저·파·패]

| 약자 | 풀이 | 의미 |
|---|---|---|
| **캔** | 캔들 | 일봉/분봉의 형태 (장대양봉, 도지, 망치형 등) |
| **거** | 거래량/거래대금 | 수급 강도. 평소 대비 1000%+, 거래대금 500억+ |
| **이** | 이평선 | 단기(3/5/10/20) + 장기(120/240/480) |
| **추** | 추세 | 주도 테마, 대장주, 상대강도 |
| **지** | 지지 | 단기이평, 옛 저항이 지지로 바뀐 자리(저지바) |
| **저** | 저항 | 박스권 상단, 전고점, 52주/사상 신고가 |
| **파** | 파동 | 분봉 파동 저점/고점, 이중바닥(W) |
| **패** | 패턴 | 컵앤핸들, 역헤드앤숄더, 박스권, 매물 소화 |

## 2. 3대 핵심 패턴 (자동 분류)

### 패턴 A — 매물 소화 후 돌파 '천양봉' (돌파매매)

**셋업**:
- 박스권 상단/52주 신고가/사상 최고가 자리에서 매물 소화
- **대량거래** 동반 장대양봉으로 강력하게 돌파
- 종가까지 매수세 살아있음

**대표 사례**:
- **SK하이닉스 (2024-02-22)** — 24년 최고가 돌파 첫 양봉
- **LG CNS (2024-05-14)** — 바닥 박스 돌파 + AM 최고가 재돌파
- **셀바스AI (2023-07-31)** — 2,300억 대량거래 박스권 돌파
- **에코프로 (2024-02-07)** — 컵앤핸들 완성 + 사상최고가
- **위메이드 (2024-02-16)** — 바닥권 첫 상한가 3,770억 대금

**진입 조건**:
- S1 (박스권 돌파) + S2 (거래량) + S3 (장대양봉) + S5 (전고점) 점수 합산 ≥ 40

### 패턴 B — 급등 후 첫 눌림목 (눌림목매매)

**셋업**:
- 상한가 또는 대량거래 장대양봉 이후 첫 조정
- **거래량 급감** + 하락폭 얕음 = 매도 압력 약화
- 3일선/5일선/10일선/저지바 자리에서 도지/망치형

**대표 사례**:
- **삼성전자 (2024-05-07~08)** — 3일선/5일선 쌍도지 + 분봉 W
- **넥스턴바이오 (2024-10-07)** — 5일선 분봉 이중바닥
- **성우하이텍 (2024-10-07)** — 3일선 + 저지바 정배열 전환
- **미래생명자원 (2024-04-11)** — 10일선 + 480일선 중첩 쌍도지
- **현대로템 (2022-07-21)** — 추세선 돌파 + 5일선 양봉

**진입 조건**:
- S7 (눌림목 셋업) + S4 (이평 정배열) + S6 (과열 회피) 점수 합산 ≥ 40

### 패턴 C — 분봉 이중바닥 + 정배열 수렴 (분봉 눌림목)

**셋업**:
- 일봉 지지 라인 도달 + **분봉상 이중바닥(W) 형성**
- 분봉 이평선이 한곳으로 수렴 → 정배열 전환
- 장 후반부(2:30 이후) 저점 점진 상승

**대표 사례**:
- **NPC (2021-12-24)** — 5일선 + 분봉 정배열 수렴
- **AP위성** — 상한가 후 3일선 + 분봉 정배열

### 패턴 D — 장기이평 돌파 (대시세 초입)

**셋업**:
- 120일/240일/480일선이 우하향 → 횡보로 전환
- **대량거래** 동반 장기이평 돌파
- 지지를 받으며 재상승

**대표 사례**:
- **씨젠, 신풍제약** — 코로나 테마 대시세
- **금양 (2023-07-25)** — 2차전지 후행주 대시세
- **미래생명자원** — 240일/480일선 동시 돌파

## 3. 실전 종가매수 5단계 프로세스

### Step 1 [확인] — 진짜 돌파 여부 (3:00 PM ~)
```
✓ 거래대금 500억+ (이상적으로 1000억+)
✓ 박스권/52주/사상최고가 돌파 확인
✓ 매물대 장악 (윗꼬리 짧음)
```

### Step 2 [조정] — 건강한 눌림목 판별 (3:10 PM ~)
```
✓ 직전 N일 거래량 급감 (매도 둔화)
✓ 음봉 길이 짧음, 하락 완만 (이쁜 조정)
✗ 폭락이나 급격한 매물 출회 ❌
```

### Step 3 [구간] — 지지선 도달 확인
```
✓ 3일선/5일선/10일선 부근
✓ 또는 저지바(옛 저항→지지) 자리
✓ 또는 단기+장기 이평 중첩 자리 (강한 지지)
```

### Step 4 [신호] — 분봉 최종 시그널 (3:20 PM ~)
```
✓ 분봉 이평선 수렴 → 정배열 전환 초기
✓ 분봉 이중바닥(W) 패턴
✓ 하단 아래꼬리 도지/망치형 캔들
✓ 장 후반부 매수세 유입
```

### Step 5 [진입] — 분할 매수 (3:25 ~ 3:30 PM)
```
✓ 종가 분할 매수 (한 번에 풀 매수 ❌)
✓ 종목당 자금 비중 10~15%
✓ 애프터마켓(시간외) 강세 확인 시 추가 매수
✓ 비중 조절은 종가 안착 강도에 비례
```

## 4. 청산 룰

### 다음날 시초 시나리오

| 시나리오 | 대응 |
|---|---|
| 갭상승 +1.5%↑ | 시초 1/3 청산 → 트레일링 -2% → 추가 분할 |
| 보합 ±1% | 장중 고가 1/2 청산 → 눌림목 지지 시 잔여 보유 |
| 갭다운 -1%↓ | 즉시 시초 손절 → 반등 시 본전 청산 |
| 1차 슈팅 +10% | 1/2 즉시 청산 → 잔여 눌림 확인 |

### 손절 (필수)
- **3일선/5일선 종가 이탈** = 기계적 손절 (하승훈식)
- **매수가 -3% 도달** = 무조건 손절
- **분봉 파동 저점 이탈** = 즉시 손절
- **T+2일 종가** = 무조건 청산 (전략 원칙)

## 5. 회피해야 할 종목

```
✗ 직전 5일 누적 +25%↑ (과열, 슈팅 끝물)
✗ 윗꼬리/몸통 > 1.5 (매도 압력)
✗ 거래량 감소하면서 상승 (수급 약함)
✗ 동전주 (500원 미만)
✗ 상장 6개월 미만 (변동성 과대)
✗ 거래정지/관리종목 이력
✗ 2등주/3등주 (대장주가 아님)
```

## 6. HTS 검색식 (수식 관리자)

### 6.1 강세 종목 포착 (기준봉)
```
A = avg(v, 20)
조건: V > A(1) * 10
  AND C > O * 1.12
  AND V * C > 50,000,000,000
설명: 20일 평균 대비 거래량 10배+ 상승률 12%+ 거래대금 500억+
```

### 6.2 대시세 초입 (장기이평 돌파)
```
조건: Crossup(C, avg(C, 120))
  AND V > V(1) * 5
설명: 종가가 120일선을 대량거래(전일 5배+)로 상향 돌파
       (240, 480일선도 동일 적용 가능)
```

### 6.3 조건검색기 기본 세팅
```
□ 관리종목/우선주/ETF/스팩 제외
□ 거래대금 500억+ (상위 50위 이내)
□ 250일 신고가 경신 또는 5% 이내 근접
□ 주가등락률 10%+ 상승 중
□ 시가총액 2,000억+
```

## 7. 승률 90%를 만드는 4가지 필수 조건

성공률 90%를 담보하려면 **반드시 4개 모두 충족**:

1. **시장의 핵심 주도 테마** — 후행 테마는 진입 ❌
2. **대량거래 장대양봉** — 거래대금 500억+, 1000% 거래량 폭증
3. **의미있는 돌파 자리** — 박스권/신고가/장기이평 중 하나
4. **장 막판까지 살아있는 매수세** — 종가 부근에서 매도 압력 없음

이 4가지 + **테마 대장주(상대강도 강함)** = 하승훈 5조건.

## 10. 최종 명심사항

> **"예측은 하되, 대응은 기계적으로."**

- 주식에 100%는 없음 — 높은 확률에 베팅할 뿐
- 손절은 자존심이 아닌 자본 보호
- 한 번의 큰 손실 = 10번의 작은 수익보다 치명적
- 좋은 종목 + 좋은 시점이 동시에 충족돼야만 진입
- 시점이 안 맞으면 좋은 종목도 손절 (현대오토에버 사례)
