# 종가매매 통합 룰 명세서

> 본 문서는 코드(scanner.py, rules.py, watchlist.py, config.py)에 구현된 모든 매매 룰의 텍스트 사양입니다.
> 코드와 문서는 1:1 매핑되어야 합니다. 코드 수정 시 본 문서도 동기화하세요.

작성일: 2026-05-21 · 버전: v2.0 (확장)

## 0. 전략 정의

- **종가매수 (종가베팅) 전용** — 장 마감 직전 매수
- **다음 날 시초가 / 장중 고가 / T+2 종가** 중 하나에서 청산
- **최대 1~2일 보유** — 그 이상은 룰 위반
- **종목당 자금 비중 10~15%** — 3종목 분산 (총 30~45% 가동)
- **현금 비중 ≥ 30%** 항시 유지 (이벤트 장세 대응)

## 2. 10대 매수 시그널 (점수 0~100 각각)

### S1: 박스권 돌파 (가중치 25)
**정의**: 직전 60일간 종가가 25% 이내 변동폭(횡보 박스권) 안에 있었고, 오늘 그 박스 상단을 종가가 돌파.
- 박스권 + 돌파 → 50~100점
- 박스권 + 95% 근접 → 40점
- 박스권 아님 + 돌파만 → 30점
- 그 외 → 0점

**의미**: 하바로셀의 "힘의 응축". 매물 소화 끝나고 신규 매수세 유입.

### S2: 거래량 폭증 (가중치 25)
**정의**: 당일 거래량이 직전 20일 평균의 N배 이상.
- 3배 이상 → 50~100점
- 그 미만 → 비례 감점

**의미**: "수급 선점". 세력 진입의 가장 강력한 흔적.

### S3: 장대양봉 (가중치 12)
**정의**: 시가 → 종가 +5% 이상 상승.
- +5% → 50점, +15% → 100점
- 윗꼬리/몸통 > 1.5 → **50% 감점** (매도 압력)

**의미**: 강한 매수세. 종가까지 밀어올린 의지.

### S4: 이평선 정배열 (가중치 12)
**정의**: 종가 > MA5 > MA20 > MA60.
- 종가 > MA5 → 25점
- 종가 > MA20 → 25점
- 종가 > MA60 → 25점
- MA5 > MA20 > MA60 정배열 → 25점

**의미**: 추세 전환 또는 상승 추세 지속 확인.

### S5: 전고점 돌파/근접 (가중치 12)
**정의**: 종가가 직전 60일 고점을 돌파했거나 95%+ 근접.
- 돌파 → 70~100점
- 95~100% 근접 → 0~70점 비례

**의미**: 매물대 돌파. "신고가는 가는 종목이 간다".

### S6: 과열 회피 (가중치 5)
**정의**: 직전 5일 누적 상승률이 낮을수록 점수 ↑.
- 누적 0% 이하 → 100점
- 누적 25% 이상 → 0점

**의미**: "차분히 조정 거친 종목". 이미 끌어올린 종목은 슈팅 끝물.

### S7: 눌림목 셋업 (가중치 5)
**정의**: 최근 20일 중 +10% 이상 슈팅이 있었고, 그 후 MA20 근처까지 조정, 오늘 양봉으로 반등 시작.
- 반등 시작 (저점 대비 +2%↑) → 80점
- 조정만 진행 중 → 40점
- 그 외 → 0점

**의미**: 하바로셀의 "눌림목 불타기" 셋업.

### S8: 수급 연속성 (가중치 4)
**정의**: 최근 5일 평균 거래량 ÷ 직전 30일 평균 거래량 비율.
- 1.5배 이상 → 50~100점
- 1.0~1.5배 → 0~50점

**의미**: 거래량 추세가 우상향. 신규 매수세 유입 중.

### S9: 장기 이평선 돌파 (가중치 8, 하승훈)
**정의**: 120/240/480일선이 우하향→횡보로 바뀌고, 종가가 그 이평선을 돌파.
- MA120 돌파 + 횡보 → +30점
- MA240 돌파 + 횡보 → +35점
- MA480 돌파 + 횡보 → +35점
- 합산 최대 100점

**의미**: 하승훈의 "대시세 초입" 시그널. 장기 하락 추세를 뚫는 매물 소화 완료.

### S10: 상대강도 (가중치 8, 하승훈)
**정의**: 종목 20일 수익률 - 벤치마크(KOSPI 지수) 20일 수익률.
- +30%p 이상 → 100점
- +15~30%p → 70~100점
- +5~15%p → 40~70점
- 0~5%p → 0~40점

**의미**: 하승훈의 "주도주 판별". 시장보다 강한 종목이 진짜 대장주.

### BONUS: 워치리스트 (최대 +5점)
- 하바로셀 검증 종목 → +50% 보너스
- 하승훈 분석 종목 → +50% 보너스
- 사용자 매매 종목 → +30% 보너스
- 테마 분류 종목 → 테마당 +15% (최대 2개 테마)

## 3. 진입 4단계 (rules.py)

### Stage 1 — 1차 필터 (3:00 PM ~)
```
✓ 거래대금 ≥ 50억
✓ 시가총액 ≥ 2000억
✓ 등락률 7~29% (상한가 30% 제외)
✗ 직전 5일 누적 +25% 이상 = 과열 (제외)
```

### Stage 2 — 차트 셋업 (3:10 PM ~)
```
✓ 종합 점수 ≥ 40 (S1~S8 가중 평균)
✗ 윗꼬리/몸통 > 1.5 = 매도 압력 (제외)
```

### Stage 3 — 3:20 PM 수급 확인 (호가창)
```
[HTS 별도 확인 필수]
✓ 호가창 매수 잔량 대량 누적
✓ 동시호가 직전 호가 단단함 (밀리지 않음)
✓ 외인/기관 매수 우위
✓ 동종 테마 종목들도 강세
```
*코드에서는 점수/거래량 비율로 프록시.*

### Stage 4 — 최종 진입 (3:25 ~ 3:30 PM)
```
✓ 위 조건 통과 → 종가로 시장가 매수
✓ 종목당 자금 비중 10~15%
✓ 한 번에 풀 매수 ❌, 분할 매수 권장
```

## 4. 청산 4시나리오 (rules.py: simulate_exit)

### A. 갭 상승 (시초가 +1.5% ↑)
```
1) 시초 30분 내 1/3 청산 (즉시 익절)
2) 트레일링 -2% 자동 설정
3) 추가 슈팅 시 1/3씩 분할 매도
```

### B. 보합 시초 (±1%)
```
1) 30분~11시 사이 고가 형성 시 1/2 청산
2) 눌림목 발생 → 전일 종가/시가 지지 확인
3) 재반등 시 잔여 청산 OR 불타기
```

### C. 갭 다운 (-1% ↓)
```
1) 동시호가 미체결 매수 금지
2) 시초가 즉시 손절
3) 매수가 -3% 도달 = 무조건 손절
```

### D. 1차 슈팅 +10% 도달
```
1) 1/2 즉시 청산 (수익 확정)
2) 잔여 50% → 눌림 지지 확인 후 보유 결정
3) 다음 날 종가까지 청산
```

### 강제 청산
- **T+2일 종가 무조건 청산** (1~2일 보유 원칙)
- **상한가 진입 시** 1/2 즉시 청산 (체결 꼬임 회피)

## 8. 자금 관리 (Position Sizing)

```
초기 자본 1,000만원 가정

→ 종목당 비중: 10~15% (100~150만원)
→ 동시 보유 최대 3종목
→ 가동 자금: 30~45%
→ 현금 비중: 55~70%

손절 발동 시:
  → 매수가 -3% = 종목당 최대 손실 3~4.5만원 (전체의 0.3~0.45%)
  → 3종목 동시 손절 시 최대 손실 9~13.5만원 (전체의 1%대)
```

## 9. 평가 지표 (백테스트)

| 지표 | 합격선 | 비고 |
|---|---|---|
| 승률 | ≥ 55% | 50% 미만이면 룰 재검토 |

| 손익비 | ≥ 1.5 | 익절 vs 손절 평균 비율 |
| 샤프 비율 | ≥ 1.0 | 위험조정수익 |
| 최대 손실 | ≤ -5% | 단일 거래 |

## 10. 주의사항

1. **본 룰은 통계적 보조** — 매수/매도 시그널 절대 보장 아님
2. **하바로셀 강의 자체 미시청** — 자료 텍스트 기반 추정. 실제 강의 뉘앙스 차이 가능
3. **선택 편향** — 분석에 사용된 차트 27장은 성공 사례 위주. 실패 패턴 회피 필터 추가 필요
4. **3:20 PM 호가창 분석은 HTS 영역** — 코드에서는 점수 프록시
5. **시장 환경 변화** — 베어마켓에서는 룰 자체가 작동 안 할 수 있음
6. **반드시 페이퍼 트레이딩 최소 4주 후 실전 적용**
