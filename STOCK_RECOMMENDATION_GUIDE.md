## 1️⃣ SuperScore 정의

```
SuperScore = p_sw × 5 + p_100+ × 2 + p_50+ × 1 - p_loss × 3

p_sw      : RF 모델이 예측한 슈퍼위너(peak ≥ 200%) 확률
p_100+    : 100% 도달 확률
p_50+     : 50% 도달 확률
p_loss    : 손절(-20%↓) 확률
```

**왜 이 가중치?**
- 100조합 그리드 서치로 OOS 4년 최고 수익률 조합
- 슈퍼위너 강조 (×5) → 큰 수익 잡기
- 손절 페널티 (×3) → 위험 제어
- 100%+ 보조 (×2) → 중간 수익 보완

## 5️⃣ 매수 룰

### 매수 시점
- **1순위**: 시그널 발생일 **NXT 19:50 시장가**
- **2순위**: 다음 영업일 (D+1) 09:00 정규장 시초가
- 두 옵션 차이 0.7%p (시뮬 +85.2% vs +84.5%)

### 매수 종목 선정
```
1. cache/today_picks.csv 확인 (자동 갱신)
2. SuperScore TOP 5 매수
3. 등급 ★ 강력매수 우선
4. ⚠️ 손절위험 등급은 매수 X
5. 주 누적 5건 채우면 그 주 stop
6. 시그널 0건이면 매수 X (현금 보유)
```

### 매도 시점
- **매수일 + 180거래일 후 정규장 종가 시장가**
- 익절/손절 룰 X (그냥 묻기)

### 자본 배분
- **종목당 10만원 고정**
- 자본 1억 → 종목당 0.1%
- 동시 보유 평균 130건 (자본 13% 사용)
